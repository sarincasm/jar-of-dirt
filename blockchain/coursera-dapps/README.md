# Developing Dapps with Truffle

[Decentralized Applications (Dapps)](https://www.coursera.org/learn/decentralized-apps-on-blockchain)

## Command Cheat Sheet

-  `truffle init`
-  `truffle develop`
-  `truffle migrate --reset`
-  `truffle test`
