# CSS

Code based on the course [CSS for JavaScript Developers](https://css-for-js.dev/) by [Josh W Comeau](https://twitter.com/JoshWComeau)

## Quick Start

```bash
npm install
npm run dev
```

Visit [http://localhost:2000](http://localhost:2000)

## Initial Setup

```bash
npx create-next-app@latest
```
