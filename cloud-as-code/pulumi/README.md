# Pulumi - Deploying and S3 Website using TS

This is a simple example of how to deploy a static website to S3 using Pulumi and Typescript.

## Getting Started

```bash
pulumi new aws-typescript
pulumi config set aws:profile eels.de.main
pulumi up
```
